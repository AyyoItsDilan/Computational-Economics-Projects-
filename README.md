# Computational-Economics-Projects-
ECON 380 is an economics class at Case Western Reserve University focusing on Computational Economics. Collection of projects applying numerical methods, simulation, and agent-based modeling to economic and social science problems. Covers market dynamics, epidemic modeling, optimization, and growth theory using Python.

Project 2: Implements the Gale-Shapley algorithm to solve the stable matching problem. Simulates matching between two groups (e.g., men and women) based on preference rankings and produces a stable outcome with no blocking pairs.

Project 3: Simulates the spread of a virus using the SIR (Susceptible–Infected–Recovered) model. Tracks infection dynamics over time and visualizes how populations transition between states under fixed transmission and recovery rates.

Project 4: Extends the SIR model to multiple regions using an origin-destination (OD) matrix. Models how movement between regions affects disease spread and evaluates policy interventions like travel reduction and transmission control.

Project 5: Agent-based simulation of a market using object-oriented programming. Models consumers and producers with heterogeneous preferences and costs to determine equilibrium price and quantity, including shifts from technological changes.

Project 6: Simulates a public goods (matching fund) game using agent-based modeling. Compares behavior across player types (free riders, conditional cooperators, strong cooperators) and analyzes how cooperation evolves over repeated interactions.

Project 7: Applies numerical methods (Newton-Raphson, SciPy optimization) to solve root-finding and maximization problems. Includes economic applications such as utility maximization and visualization of indifference curves and budget constraints.

Project 8: Implements the Solow-Swan economic growth model using a class-based structure. Simulates capital accumulation, income growth, and steady-state equilibrium, with policy experiments on savings rates.

Project 9: Analyzes NBA game and player statistics using Python (Pandas, NumPy). Cleans real-world datasets, handles missing data, and extracts insights such as top-performing players and highest-scoring games. Includes filtering by date, ranking performance metrics, and basic exploratory data analysis.
